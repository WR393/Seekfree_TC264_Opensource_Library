# Frontend Development Guidelines

> Not applicable for the current repository state.

---

## Status

This repository currently contains embedded firmware only. The active source tree is centered on:

- `user/`
- `code/`
- `libraries/`

There is no web UI, desktop GUI, mobile app, or frontend package to standardize today.

---

## Guidance

- Do not force firmware code into frontend concepts just to satisfy the Trellis scaffold.
- If a host-side tuning UI, telemetry dashboard, or configuration tool is added later, create real frontend conventions based on the actual code and framework.
- Until that happens, the `backend` guidelines are the authoritative rules for the current project.
