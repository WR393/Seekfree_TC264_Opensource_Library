# Seekfree TC264 Opensource Library

基于逐飞 `TC264 Opensource Library` 的单轨车平衡控制工程。

当前工程主要用于：
- TC264 平台下的舵机 / 无刷电机 / IMU 联调
- 单车平衡控制实验
- 启动、平衡、方向约束与实车调参验证

## 项目说明

这个仓库是在逐飞 TC264 开源库基础上做的实车控制工程，核心目标是实现单轨车在固定前进速度下通过舵机转向完成动态平衡。

当前代码中已经包含：
- 舵机控制
- 无刷电机速度控制
- IMU660RC 姿态读取
- 串级 PID 平衡控制
- 运行时舵机中位微调
- 平衡调试日志与调参相关 spec 记录

## 目录结构

- `user/`
  - 主函数、中断服务、系统入口
- `code/app/`
  - 平衡、IMU、电机、舵机等应用层逻辑
- `code/driver/`
  - PID、电机驱动封装等
- `libraries/`
  - 逐飞库与 Infineon TC264 底层库
- `.trellis/spec/`
  - 本项目调试过程沉淀出的规范和经验

## 当前状态

目前工程已经完成以下阶段：
- 模块级测试通过：舵机、电机、IMU
- 小车可实现一定时间的平衡
- 已支持舵机中位在线调节
- 已开始区分启动阶段、平衡阶段和方向约束阶段

仍在继续优化的部分：
- 落地发车稳定性
- 比赛场地橡胶跑道上的平衡成功率
- 航向保持与蛇形轨迹抑制
- 舵机摆动幅度与控制平顺性

## 开发环境

- MCU: Infineon TC264
- Library: Seekfree TC264 Opensource Library
- 工程类型: AURIX / ADS 工程

## 关键文件

- `user/cpu0_main.c`
  - 主循环、启动逻辑、按键微调入口
- `user/isr.c`
  - IMU 与各控制环周期调用
- `code/app/balance_app.c`
  - 平衡主控制逻辑
- `code/app/imu_app.c`
  - 姿态角与角速度处理
- `code/app/motor_app.c`
  - 电机速度读取与控制
- `code/app/servo_app.c`
  - 舵机 PWM 输出

## 备注

这个仓库目前以实车调试和控制验证为主，不是通用成品方案。参数、舵机中位、启动策略、方向约束强度都需要结合具体车体结构和实际场地继续调整。
